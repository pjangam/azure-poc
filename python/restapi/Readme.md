# This is rest api to manage aks cluster on azure

## Tech stack:
python, flask, virtualenv

## Environment variables:

- AZURE_CLIENT_ID
- AZURE_CLIENT_SECRET
- AZURE_TENANT_ID
- AZURE_SUBSCRIPTION_ID

## How to run

make run
